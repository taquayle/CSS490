Here is the specification for the assignment.

         You will begin with the API from Example 4.2 (4.2.KeyboardSupport).

         You will create a new MyGame.js test case and modify (very slightly) the API to support/export new functionality.

         MyGame test case:

o    The world: canvas: 640x480, world: 100x75

o    Your app will draw the Red Cursor initially at the center of the viewport.

  Red Cursor: size 1x1

o    Arrow keys: (Left/Right/Up/Down) moves the Red Cursor accordingly.

o    Space bar: creates n squares where:

  n is a random number between 10 to 20

  The center of the squares are randomly distributed within 5 units of the Red Cursor.

  The size of each square is a random number between 1 to 6.

  The squares are randomly rotated.

o    D-key: the clicking of the D-key will send your app into delete mode. Your app will remind in delete mode until all the created squares are deleted.

  The delete mode: will cause the squares to be deleted in the order and according to the time-interval of their creation. E.g.,

         Set-1: at time t0, you pressed the first space-bar and squares Set-1 is created

         Set-2: 3-seconds after t0, you press the space bar again and square Set-2 is created

         Set-3:10-seconds after t0, you press the space bar again and square Set-3 is created

  Now, the deletion order will follow the creation order, that is,

         Delete Set-1, when D-key is clicked

         3-seconds after t1, you will delete Set-2

         10-seconds after t1, you will delete Set-3

  When all squares are deleted, your application will exit from delete mode.

         API modification (very slightly):

o    Engein_GameLoop.js: you will examine the core of Game Loop and decide what are the important information to pass out so that you can report on

  Time to call update/draw

  Number of frames drawn in a second

  Number of update() calls per each draw() function call

o    Renderer.js: this is where I modify the API to include a relative object creation time, this relative time is used to support time-ordered object deletion.

         HTML page text output

o    You will modify index.html to support text of information concerning your test case (e.g., number of squares on screen) and the engine state (re-draw frame rate).

c.        When removing elements from an array

                                                                   i.      I iterate from the back to the front

for (i=myArray.length-1; i>=0; i--)

 

                                                                 ii.      To delete, I call:

myArray.splice(i, 1); // remove from position i, one element

4.       You will need to call Javascript: Math.random() to work with random numbers. Remember:

a.       This function returns a number between 0 to 1

b.       To create a random number between A and -A, you can do this:

A * (Math.random() 0.5)

c.        To create a random number between X and Y, you can do this:

X + (Y * Math.random())
